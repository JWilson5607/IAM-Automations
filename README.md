# IAM Automations

PowerShell scripting and module work centered around Identity and Access Management (IAM) processes and workflows.

## Overview

This repo contains automation scripts and reusable PowerShell modules for managing identities, access controls, and related IAM operations. Work here targets common IAM tasks such as user provisioning, deprovisioning, role/group management, and access reviews.

## Structure

- **Scripts** — standalone PowerShell scripts for specific IAM tasks
- **Modules** — reusable PowerShell modules supporting broader IAM workflows

## Requirements

- PowerShell 5.1+
- Appropriate permissions for the target IAM systems (e.g., Active Directory, Azure AD)
